# Contributing

This code is generated from a private monorepo at Chronary. Pull requests opened here are welcome as proof-of-concept but cannot be merged directly — any change would be overwritten on the next generated release. To propose a change to the source of truth, open an Issue here describing the intended behavior and the Chronary team will route it into the monorepo.

README and documentation contributions opened here will be considered and re-applied in the monorepo when accepted.

For bug reports and feature requests, please use Issues on this repository.
